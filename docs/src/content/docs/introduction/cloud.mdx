NativeLink Cloud is the quickest and simplest way to get started with NativeLink.

NativeLink Cloud provides robust cloud solutions that enable remote caching for your builds
with just a few additions to your .bazelrc.
This can significantly speed up your build times and improve your development workflow.

## Getting Started
Get up and running quickly with our [fully managed NativeLink Cloud service](https://app.nativelink.com).
It's **free** for all individuals and teams.

If you encounter any issues or have any questions about using NativeLink Cloud,
don't hesitate to reach out to us on [Slack](https://nativelink.slack.com/join/shared_invite/zt-281qk1ho0-krT7HfTUIYfQMdwflRuq7A#/shared-invite/email).
